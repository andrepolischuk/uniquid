# Uniquid

  Unique ID generator

## Instalation

  Browser:

```html
<script src="https://cdn.rawgit.com/andrepolischuk/uniquid/1.0.2/uniquid.min.js"></script>
```

  Component(1):

```sh
$ component install andrepolischuk/uniquid
```

  Npm:

```sh
$ npm install uniquid
```

## API

### uniquid([prefix])

  Generate ID

```js
uniquid();
// atquj2qdjig4

uniquid('user_');
// user_atqujc5gjj4g
```
