# Ccc

  Colors converter

## Instalation

  Browser:

```html
<script src="https://cdn.rawgit.com/andrepolischuk/uniquid/1.0.0/dist/uniquid.min.js"></script>
```

  Component(1):

```sh
$ component install andrepolischuk/uniquid
```

  Npm:

```sh
$ npm install uniquid
```

## API

### uniquid()

  Generate ID

```js
uniquid();
// 8cjlrebouy048o
```

### uniquid(prefix)

  Generate ID with prefix

```js
uniquid('user_');
// user_8cjlrixab2g4go
```
